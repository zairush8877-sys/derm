# Aura — Android-приложение для RuStore (WebView-обёртка)

Тонкое Android-приложение, которое показывает сайт **aura-wellness.ru** как
нативное приложение: своя иконка `aura.`, бежевый сплэш-экран, доступ к камере
(сканы кожи и еды), выбор фото из галереи, аппаратная кнопка «назад» листает
историю сайта. Все обновления сайта сразу видны в приложении — пересобирать
не нужно.

## Что это даёт
- Приложение в **RuStore** (публикация физлицом — бесплатно, ИП не нужен)
- Иконка Aura на домашнем экране, запуск без браузерной строки
- Камера для AI-сканов работает как в обычном приложении

## Как собрать APK/AAB (нужен компьютер с Android Studio)

1. Установите **Android Studio** (бесплатно, android.com/studio).
2. Откройте папку `mobile/` как проект (File → Open).
3. Дождитесь синхронизации Gradle (сама скачает зависимости).
4. Соберите:
   - для теста: **Build → Build APK** → получите `app-release.apk`, ставится на телефон;
   - для RuStore: **Build → Generate Signed Bundle** → `.aab`, подписанный ключом.
5. Ключ подписи создаётся один раз в мастере (Create new… ) — **сохраните
   файл .jks и пароли**, они нужны для всех будущих обновлений.

Из командной строки (если установлен Android SDK):
```bash
cd mobile
./gradlew assembleRelease        # APK
./gradlew bundleRelease          # AAB для стора
```

## Публикация в RuStore
1. Зарегистрируйтесь на **console.rustore.ru** (можно как физлицо).
2. Создайте приложение → загрузите `.aab`.
3. Заполните карточку: название Aura, иконка, описание, скриншоты (можно снять
   с работающего приложения), возрастной рейтинг, категория «Здоровье».
4. Отправьте на модерацию (обычно 1–3 дня).

## Важно про домен
Приложение открывает `https://aura-wellness.ru/` (см. `START_URL` в
`MainActivity.kt`). Пока домен не делегирован, для теста можно временно
заменить START_URL на текущую рабочую ссылку туннеля.

## Структура
```
mobile/
├── settings.gradle.kts, build.gradle.kts   — конфигурация Gradle
├── app/build.gradle.kts                     — зависимости, версия, applicationId
└── app/src/main/
    ├── AndroidManifest.xml                  — разрешения (интернет, камера)
    ├── java/ru/aura/wellness/MainActivity.kt — WebView + камера + «назад»
    └── res/                                  — тема, сплэш, иконки (палитра «Спа»)
```

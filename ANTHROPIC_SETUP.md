# Подключение реального AI (ключ Anthropic) — 10 минут

Сейчас Aura работает в демо-режиме (детерминированные результаты). Один ключ
переключает всё на настоящий AI: анализ кожи и еды — **Claude Opus 4.8** (vision),
ассистент — **Claude Haiku 4.5**.

## Шаг 1. Получить ключ

1. Зайдите на **console.anthropic.com** и создайте аккаунт (email/Google).
2. Раздел **Billing → Buy credits** — пополните баланс (минимум $5;
   на старте этого хватит на ~100–500 фото-анализов).
3. Раздел **API Keys → Create Key** — скопируйте ключ вида `sk-ant-api03-...`.
   ⚠️ Ключ показывается один раз — сохраните его.

> **Для РФ:** для оплаты кредитов нужна зарубежная банковская карта
> (виртуальные карты казахстанских/киргизских банков подходят). Сам сервер
> может находиться где угодно — API работает по HTTPS.

## Шаг 2. Вставить ключ

**Локально:** в файле `.env`:
```
ANTHROPIC_API_KEY=sk-ant-api03-ваш-ключ
```

**На VPS (docker):**
```bash
nano /root/aura/.env          # вписать ANTHROPIC_API_KEY=...
docker compose up -d          # перезапустить
```

Демо-режим выключится автоматически — отдельного переключателя не нужно
(`DERM_MOCK_MODE` должен быть пустым или `0`).

## Шаг 3. Проверить (3 способа)

```bash
# 1) Из терминала — самопроверка обеих моделей (стоит доли цента):
python -m app.cli aitest

# 2) Быстрый статус:
curl localhost:8000/health
# → "mode": "Реальный AI (Claude vision)"
```

3) Или в админке `/admin` — кнопка **«🤖 Проверить AI»**: покажет результат
пробного чат- и vision-вызова и подсказку, если что-то не так.

## Стоимость (напоминание из COSTS.md)

| Операция | Модель | ≈ стоимость |
|---|---|---|
| Фото-анализ кожи/еды | Opus 4.8 | $0.05–0.09 ≈ 4–8 ₽ |
| Ответ ассистента | Haiku 4.5 | ~$0.003 ≈ 0,3 ₽ |

Скан продаётся за 199 ₽ → маржа 93–96%. Модели меняются в `.env`:
`DERM_MODEL` (vision), `DERM_CHAT_MODEL` (чат) — например, `claude-sonnet-5`
для vision снизит стоимость скана до ~2 ₽ с небольшой потерей качества.

## Безопасность

- Никогда не коммитьте ключ в git (`.env` уже в `.gitignore`).
- Если ключ засветился — отзовите его в консоли (Delete) и создайте новый.
- В консоли Anthropic можно поставить **Spend limit** — месячный потолок
  расходов, чтобы исключить сюрпризы.

## Частые ошибки

| Симптом | Причина |
|---|---|
| `authentication_error` | Опечатка в ключе / ключ отозван |
| `Your credit balance is too low` | Нет кредитов — пополните Billing |
| Режим остался «Демо» | Не перезапустили приложение после правки `.env` |

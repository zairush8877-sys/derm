# derm — AI-анализ кожи (3-в-1 + AI-трекер)

Дерматологически обоснованная платформа AI-анализа кожи. Один продукт объединяет
четыре сценария:

1. **B2B API** — лицензируемый REST-эндпоинт (фото → анализ кожи в JSON) для
   beauty-брендов, ритейлеров и телемед-платформ. Биллинг по сканам.
2. **Веб-демо** — страница: загрузил фото → разбор кожи → PDF-отчёт с
   рекомендациями. То, что показывают брендам как proof-of-concept.
3. **Движок персональных протоколов** — фото (+ квиз) → персональный уход
   утро/вечер, с датой обновления через 30 дней.
4. **AI-трекер** — сохраняет сканы во времени и показывает динамику кожи: что
   улучшилось, а что требует внимания.

> ⚠️ **Важно:** это **косметический анализ кожи** для подбора ухода, **а не
> медицинский диагноз**. Не заменяет очную консультацию врача-дерматолога.
> Формулировка зафиксирована в ответах API, PDF и интерфейсе.

## Архитектура

```
app/
  main.py            FastAPI: B2B API + демо + трекер
  config.py          настройки и авто-демо-режим
  schemas.py         Pydantic-схемы (общий контракт)
  analysis/          движок анализа (Claude vision + mock-fallback)
  protocol/          генерация персонального протокола ухода
  tracker/           AI-трекер динамики кожи
  report/            PDF-отчёты (fpdf2, кириллица через DejaVu)
  api/               B2B API v1 + аутентификация по X-API-Key
  db/                хранилище SQLite
  static/            фронтенд: index.html, tracker.html
tests/               pytest (всё в демо-режиме, без сети)
```

## Режимы работы

- **Реальный AI** — если задан `ANTHROPIC_API_KEY`, анализ выполняет
  Claude vision по фото.
- **Демо-режим (mock)** — без ключа платформа работает на детерминированных
  данных: весь сценарий (анализ → PDF → трекер) проходит без затрат на API.

## Запуск

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Затем откройте:

- `http://localhost:8000/` — веб-демо (анализ + PDF)
- `http://localhost:8000/tracker` — AI-трекер
- `http://localhost:8000/docs` — Swagger-документация B2B API

### Конфигурация

Скопируйте `.env.example` в `.env` и при необходимости задайте `ANTHROPIC_API_KEY`.

## B2B API

Заголовок `X-API-Key` обязателен (демо-ключ см. в `.env.example`).

```bash
# Анализ кожи
curl -H "X-API-Key: demo-key-derm-2026" \
     -F image=@face.jpg \
     http://localhost:8000/v1/analyze

# Персональный протокол
curl -H "X-API-Key: demo-key-derm-2026" \
     -F image=@face.jpg -F age=30 \
     http://localhost:8000/v1/protocol

# Использование (для биллинга по сканам)
curl -H "X-API-Key: demo-key-derm-2026" http://localhost:8000/v1/usage
```

## Тесты

```bash
pytest
```

Все тесты выполняются в демо-режиме, без обращения к сети.

# derm — wellness super-app (🇷🇺) + AI-анализ кожи и еды

Единое приложение для российского рынка: **маркетплейс красоты и здоровья**
(типа «Золотого яблока») со встроенными AI-инструментами. Один продукт:

1. **Магазин** — уходовая косметика, гаджеты и спортпит (цены в ₽), фильтр по
   категориям, корзина. Товары рекомендуются под результат анализа кожи.
2. **AI-анализ кожи** — фото → разбор кожи → персональный протокол ухода + PDF.
   **Фото-анализ платный** (кредиты за скан), протокол по квизу — бесплатно.
3. **AI-трекер еды** — фото блюда → калории и БЖУ, дневник и подсчёт калоража
   за день. Фото-анализ тоже платный (AI-скан).
4. **AI-трекер кожи** — динамика по сканам во времени (что улучшилось/ухудшилось).
5. **B2B API** — лицензируемый REST-эндпоинт анализа кожи для брендов.

**Монетизация:** фото-анализ (кожа/еда) платный — 1 кредит за скан. Новому
пользователю даётся пробный скан. Слой оплаты провайдеро-независимый: демо-режим
из коробки + готовая точка подключения **ЮKassa** (`DERM_PAYMENT_PROVIDER`).

> ⚠️ **Важно:** это **косметический анализ кожи** для подбора ухода, **а не
> медицинский диагноз**. Не заменяет очную консультацию врача-дерматолога.
> Формулировка зафиксирована в ответах API, PDF и интерфейсе.

## Архитектура

```
app/
  main.py            FastAPI: страницы + все роутеры
  config.py          настройки, авто-демо-режим, цены/биллинг
  schemas.py         Pydantic-схемы анализа кожи
  analysis/          движок анализа кожи (Claude vision + mock-fallback)
  protocol/          генерация персонального протокола ухода
  tracker/           AI-трекер динамики кожи
  report/            PDF-отчёты (fpdf2, кириллица через DejaVu)
  billing/           кредиты за скан + оплата (demo + точка под ЮKassa)
  shop/              каталог, корзина, рекомендации (маркетплейс РФ)
  food/              AI-трекер еды: фото -> калории/БЖУ, дневник
  api/               B2B API v1 + аутентификация по X-API-Key
  db/                хранилище SQLite
  static/            фронтенд: index / food / shop / tracker + PWA
tests/               pytest (всё в демо-режиме, без сети)
```

## Страницы

- `/` — анализ кожи (платный фото-скан) + протокол
- `/food` — AI-трекер еды (калории/БЖУ)
- `/shop` — магазин (косметика, гаджеты, спортпит)
- `/tracker` — динамика кожи
- `/docs` — Swagger B2B API

## Режимы работы

- **Реальный AI** — если задан `ANTHROPIC_API_KEY`, анализ выполняет
  Claude vision по фото.
- **Демо-режим (mock)** — без ключа платформа работает на детерминированных
  данных: весь сценарий (анализ → PDF → трекер) проходит без затрат на API.

## Запуск

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Затем откройте:

- `http://localhost:8000/` — веб-демо (анализ + PDF)
- `http://localhost:8000/tracker` — AI-трекер
- `http://localhost:8000/docs` — Swagger-документация B2B API

### Конфигурация

Скопируйте `.env.example` в `.env` и при необходимости задайте `ANTHROPIC_API_KEY`.

## B2B API

Заголовок `X-API-Key` обязателен (демо-ключ см. в `.env.example`).

```bash
# Анализ кожи
curl -H "X-API-Key: demo-key-derm-2026" \
     -F image=@face.jpg \
     http://localhost:8000/v1/analyze

# Персональный протокол
curl -H "X-API-Key: demo-key-derm-2026" \
     -F image=@face.jpg -F age=30 \
     http://localhost:8000/v1/protocol

# Использование (для биллинга по сканам)
curl -H "X-API-Key: demo-key-derm-2026" http://localhost:8000/v1/usage
```

## Тесты

```bash
pytest
```

Все тесты выполняются в демо-режиме, без обращения к сети.

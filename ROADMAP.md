## Дизайн
Premium minimalist (эстетика WHOOP / Oura / Goop): тёплый off-white фон,
глубокий wellness-зелёный + песочный акцент, serif-заголовки, mobile-first,
адаптивная сетка. См. `app/static/styles.css`.

## B2B (основа #1)
Лицензируемый API анализа кожи + SDK (JS-виджет для сайта бренда, Python-клиент),
SaaS-тарифы с квотой сканов. См. `app/api/*`, `sdk/`.

## Что требует отдельной оценки (по «вопросам к агентству»)
- Нативные iOS/Android приложения (сейчас — PWA).
- Реальные онлайн-оплаты (ЮKassa) и доставка (СДЭК/Почта) — интеграции провайдеров.
- CRM/ERP, роли сотрудников, склад.
- Реальная дерматологическая валидация модели и мед-данные.
- Веб-push инфраструктура (VAPID), email/SMS.
